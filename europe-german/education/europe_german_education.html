Achtung: Dies war eine Phishing-Simulation

Sie haben auf eine E-Mail aus einer Sicherheitsschulung (Simulation) geklickt. Keine Sorge: Es wurden keine Daten kompromittiert. Dies ist eine gute Gelegenheit zum Lernen.

Warnsignale, auf die Sie achten sollten

  • Ein Gefühl von Dringlichkeit oder Drohungen („Ihr Konto wird gesperrt“).
  • Aufforderungen, sich über einen Link anzumelden oder Ihr Passwort zu bestätigen.
  • Die Absenderadresse stimmt nicht mit der offiziellen Domain überein.
  • Unpersönliche Anreden oder Rechtschreibfehler.

So schützen Sie sich

  • Halten Sie inne, bevor Sie klicken. Prüfen Sie den Absender.
  • Geben Sie die Adresse der offiziellen Website selbst im Browser ein.
  • Aktivieren Sie die Zwei-Faktor-Authentifizierung, wo immer möglich.
  • Melden Sie verdächtige E-Mails an Ihr IT-Team.

Faustregel: Im Zweifel nicht klicken – melden.
